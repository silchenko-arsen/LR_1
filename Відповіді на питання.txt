1. Git — розподілена система керування версіями файлів.
2. Git має три основних стани, в яких можуть перебувати файли: збережений у коміті (commited), змінений (modified) та індексований (staged).
3. Для додавання файла в репозиторій.
4. Для перевірки чи додані файли (виділені зеленим кольором – додані, червоним – не додані).
5. Для фіксації стану сховища в певний момент часу.
6. Для ініціалізації Git у папці проекту.
7. Описують, що було змінено, де змінено, чому змінено.
8. Команда git log
9. Команда git checkout дозволяє переміщатися між гілками, зробленими командой git branch.
